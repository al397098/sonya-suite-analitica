# SONYA Suite Analítica
> **Decision Support System** basado en algoritmos multicriterio de sostenibilidad fuerte y compensatoria.

---

## Concepto
**SONYA** (Sustainable Optimization Network Yield Analysis) es una herramienta avanzada para la toma de decisiones estratégicas. Permite evaluar alternativas de inversión o actuación basándose en tres ejes críticos: **Económico, Ambiental y Social**.

### Motores de Cálculo
* **Motor TOPSIS (Cerceta):** Lógica compensatoria mediante proximidad euclidiana al ideal (Algoritmo CRITIC).
* **Motor MRP (Morado):** Lógica de sostenibilidad fuerte. Implementa **protocolos de veto (Red-Line)** para garantizar mínimos innegociables.
* **Gestión BBDD:** CRUD dinámico con recálculo automático de pesos y umbrales en tiempo real.

## Instalación rápida
1. Clona el repositorio: `git clone https://github.com/al397098/sonya-suite-analitica.git`
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta el core: `python Archivos_py/sonya_api.py`
4. Abre `index.html` en tu navegador.
